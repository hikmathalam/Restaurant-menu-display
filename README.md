# Restaurant Menu Display


A React app showcasing a restaurant menu with reusable components and list rendering, built with Vite.


## Components
- **MenuItem**: Reusable component to display individual menu items with conditional vegetarian badge.

## Run Locally
1. Clone the repository
```bash
git clone https://github.com/<your-username>/restaurant-menu.git
cd restaurant-menu
```
2. Install dependencies
```bash
npm install
```
3. Start the dev server
```bash
npm run dev
```
4. Build for production
```bash
npm run build
```


## Deployment
- Deploy to **Vercel** or **Netlify**.
- On Vercel, connect your GitHub repo and use default settings; build command: `npm run build`, output directory: `dist`.
